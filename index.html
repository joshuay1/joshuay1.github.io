Artificial Intelligence / 2018

In this Artificial Intelligence project, we have implemented a python program that plays a complete game comprises of three distintictive classes: Board, Player and Piece. In a given game, a player object would interacts with the pieces on a board object. The program implements the minimax search with alpha-beta pruning to explores all the moves up to the depth of 3 and determines the best move to carry out.
